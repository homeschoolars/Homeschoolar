# Home Scholars Curriculum System (Ages 4-13)

## Age-Wise Progression

- `4-5`: Foundation
- `5-6`: Basic Understanding
- `6-7`: Core Skills
- `7-8`: Concept Building
- `8-9`: Application
- `9-10`: Analytical Thinking
- `10-11`: Problem Solving
- `11-12`: Pre-O Level Foundation
- `12-13`: O Level Readiness

- History

- Emotional Intelligence

- Financial Literacy

## Prompt Types by Difficulty

- All ages: `story`, `worksheet`, `quiz`
- Age 8+: `project`
- Age 10+: `research`, `reflection`
- Age 11+: `debate`

## API Surface

- `GET /api/age-groups`
- `GET /api/subjects?age=<age-group>`
- `GET /api/units?subjectId=<id>`
- `GET /api/lessons?unitId=<id>`
- `GET /api/content?lessonId=<id>`
- `POST /api/generate`
